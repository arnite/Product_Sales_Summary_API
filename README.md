# Product_sales_summary_API

## 1. Description

Product_sales_summary_API is an SQL based application that returns a simple report showing each product and total units sold. It features Data filtering and Aggregation.

## 2. Project Aim

The aim of this project is to reinforce the user Backend skills e.g postgreSQL aggregation, basic queries, typescript backend modelling.

## 3. Technologies Used

- **Backend**: nodejs, expressjs
- **Programming language**: typescript
- **Database**: postgreSQL, Sequelize
- **Fake Data**: fakerjs
- **Testing**: postman

## 4. API endpoints

- **GET `/api/summary`**
  _Retrieves the sales summary_
